В SETTINGS.PY проекта

Эти переменные нужны для перехода на указанные страницы (name='home' - имя указанное в пути приложения)
Необходимо при работе с 'django.contrib.auth.urls'

LOGIN_REDIRECT_URL = 'home'
LOGOUT_REDIRECT_URL = 'home'

templates/registration/login.html
Имена по умолчанию с которыми работает django.
Путь и имена обязательны для работы

 - registration
 - login.html
 {{ form.as_p }} - форма, которая так же идёт по умолчанию. (as_p каждое поле разбивает по абзацам)

Проверка на авторизацию пользователя

{% if user.is_authenticated %}
        <p>Привет {{ user.username }}!</p>
        <p><a href="{% url 'logout' %}">Выход</a></p>
      {% else %}
        <p>Зарегистрируйтесь</p>
        <a href="{% url 'login' %}">login in</a>


В тэге <form> используется кнопка <button type="submit">Login</button>
